# Content Skill

## Purpose
Create clear, useful, and brand-consistent product/editorial content optimized for discovery and conversion.

## Voice
- Warm, playful, concise.
- Helpful first, clever second.
- Avoid hype and unverifiable claims.

## Workflow
1. Identify intent: discovery, comparison, or conversion.
2. Draft structure:
   - product: hook, value, practical fit, CTA
   - blog: search intent, outline, internal links, CTA
3. Apply taxonomy consistently (emotional/intent/context tags).
4. Run SEO and readability checks.
5. Final pass for factual accuracy and tone.

## Product Copy Rules
- Keep titles clear and scannable.
- Describe real use-case and buyer fit.
- Call out constraints (space, budget, cat profile) when relevant.
- Use concrete language over generic adjectives.

## SEO Checklist
- One primary keyword per page.
- Title and H1 aligned but not duplicated verbatim.
- Meta description has clear value + CTA.
- Internal links point to relevant products/posts.
- Images have descriptive filenames and `alt` text.

## Quality Checklist
- No placeholder claims, prices, or unsupported stats.
- No duplicate tag synonyms unless intentional.
- CTA matches page intent.
- Copy is readable on mobile (short paragraphs, clear headings).

## Handoff
Provide:
- Final copy blocks and target locations.
- Required content fields for backend/CMS.
- QA checks for rendering and link correctness.
